# CritterSmash

**Celo'da Hammer NFT ile $SMASH token kazandığın whack-a-mole oyunu!**

## Özellikler
- Reown + Wagmi ile Celo wallet bağlama
- Hammer NFT kontrolü (oynamak için gerekli)
- Skor → $SMASH claim (Cloudflare backend)
- Leaderboard (on-chain)
- Farcaster Mini App uyumlu

## Kurulum
```bash
npm install
cp .env.example .env
# WalletConnect Project ID ekle
npm run dev
